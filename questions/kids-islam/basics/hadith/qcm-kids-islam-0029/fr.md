# Question

Comment s'appellent les paroles et les actions du Prophète Muhammad ﷺ que les musulmans suivent ?

## Choices

- Les Hadiths
- Les Fatwas
- Les Surahs
- Les Duas

## Answer

1

## Explanation

Les Hadiths sont les paroles, les actions et les approbations du Prophète Muhammad ﷺ. Avec le Coran, les Hadiths guident les musulmans dans leur vie de tous les jours.
