# Question

Que veut dire "Inshallah" ?

## Choices

- Merci à Allah
- Si Allah le veut
- Gloire à Allah
- Allah est grand

## Answer

2

## Explanation

"Inshallah" veut dire "Si Allah le veut". On le dit quand on parle de quelque chose qu'on espère faire dans le futur. Par exemple : "Demain j'irai à l'école, inshallah !"
